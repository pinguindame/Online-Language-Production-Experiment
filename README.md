# Online-Language-Production-Experiment

https://osf.io/uh5vr/?view_only=229679aa33604aa2a5cb400eab620996.

Link to OSF (Open Science Framework) repository containing 
1. demo files for enabling .wav-file recording time-locked to stimuli (based on jsPsych and JATOS)
2. data and analysis scripts for the manuscript "Internet-based language production research with overt articulation"

Specific README-file for the audio recording option in jsPsych can be found in the subfolder "ExperimentDemoFiles"
